# 海报样式规范（style spec）

目标：生成类似「每天60s读懂世界」的竖版可分享长图。

## 画布与排版
- 默认画布：1080×1920（可配置）
- 背景：浅色（#F7F4EF 附近），正文黑色，序号灰色
- 版式：单列 15 条（默认），每条一行/两行以内

## Header
- 主标题：每天 60s 读懂世界
- 日期：YYYY年M月D日
- 右侧：星期X（大字）
- 可选：农历/节气（若无法可靠获取，可省略）

## Body（要点）
- 序号 1–15
- 每条一句话：建议 26–40 个中文字符（允许数字/单位/引号）
- 去重：同一事件只保留 1 条，优先更权威、更具体的表述

## Footer
- 来源串：例如「新闻联播 / 人民日报 / 新华社 / Reuters / AP / BBC ...」
- 更新时间：YYYY-MM-DD HH:mm（可选）
- 署名：可选（例如 @codexcn）
