# 一句话改写规则（rewriting rules）

## 目标
把多来源新闻摘要成「一句话要点」，用于海报展示。

## 规则
- 信息密度优先：主体 + 动作 + 关键对象 + 影响（尽量含关键数字/时间）
- 不夸张：避免“震惊”“炸裂”等情绪词
- 快变事件：数字用「至少/约/据…称」避免过度精确；必要时加「截至 YYYY-MM-DD HH:mm」
- 统一口径：同一事件合并后输出 1 条，保留最重要细节
- 长度控制：默认 26–40 个中文字符；超过则删掉次要修饰语

## 模板
- 政策：机构/人物 + 发布/宣布 + 核心措施 + 生效时间/影响
- 突发：地点 + 事件 + 伤亡/影响（保守） + 最新进展
- 科技：公司/机构 + 发布/推出 + 核心指标/进展 + 行业意义
